# bat scripts
Just a collection of various bat scripts that I use because this is the simplest equivalent I've found to bash aliases and Linux style commands for Windows.

## How to use
1. Clone the repo.
2. [Add the folder to your user path](https://serverfault.com/questions/33681/how-can-i-modify-a-user-s-path-environment-variable-without-logging-out).
3. Update `config.ini`.
4. Download [Microsoft Edit](https://github.com/microsoft/edit), put it in the folder, copy it twice and name the copies `vi` & `vim`. Feel free to substitute this with a different Windows command-line based editor.
4. Profit.
